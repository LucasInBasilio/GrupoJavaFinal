# GrupoJavaFinal

<h1>Relics Tavern</h1>
<br>
    Este projeto foi criado para simular uma loja com catálogo de produtos, pagina de login/cadastro, simulação de vendas utilizando JDK8.2.
<br>
    Para executar o projeto será necessário.

<dl>
    <dt>
    JDK8.2 : necessário para executar o projeto Java.
    </dt> 
    <dt>
    NetBeans 8.2 : para desenvolvimento do projeto.
    </dt>
    <dt>
    MySQL Workbench : para escrever Skript SQL 
    </dt></dl>
    <br>
<h2>Features</h2>
<br>
O projeto pode criar clientes, deletar clientes , cadastrar produtos, efetuar vendas,verificar stock. De uma forma simples e rápida podendo servir de base para algumas 
aplicações Java.
